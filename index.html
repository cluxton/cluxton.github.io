Redux Todo List with live collaboration

Live demo   Github

A todo list with live collaboration functionality. Written using React and Redux on the front-end and Nodejs and Redis on the Backend.

Some of the reasons I enjoy working with React and Redux (Flux) are the opportunities presented by the one-way data flow. To get very basic collaboration working, all I had to do was write a Redux middleware to intercept actions and post them to a websocket.

Simple-fi

Github

A Spotify client written with Swift using the Spotify iOS SDK.

This project was born when I tried to set a playlist running on my iPad while still allowing guests to queue and skip songs. More often than not, the playlist would be stopped entirely.

The "play now" and "queue" actions in Simple-fi are much more deliberate allowing guests to change the current song without accidentally clearing the rest of the play queue. Once I have added support for user playlists, the app will be feature-complete.
